# 说明
angular + ng-zorro 构建的抽奖系统
3D球形旋转，支持bgm自定义，背景图片自定义，新增抽奖等等。

## 构建
```
npm install
```

## 运行
```
ng serve --port 0 --open
```

## 打包
```
ng build --prod
```
